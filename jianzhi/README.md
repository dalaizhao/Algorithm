# 剑指offer  66道题

## 文件

1. all : 全部集中在了一个文件Solution.java，是全部66道题的整合版本。  

2. single : 单个题的实现放在了当前目录下，对应下面目录。  

## 涉及到的数据结构与算法 
  
数据结构：数组、字符串、哈希表、二叉树、栈、队列、链表、堆    

算法：归并、动态规划、深度优先搜索（DFS）、广度优先搜索（BFS）、回溯、递归、二分查找
